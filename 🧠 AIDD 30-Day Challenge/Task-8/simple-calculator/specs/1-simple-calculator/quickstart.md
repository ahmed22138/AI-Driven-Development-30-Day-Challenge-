# Quickstart: Simple Calculator

To run the Simple Calculator application:

1.  Ensure you have Python 3.x and Streamlit installed.
2.  Navigate to the `calculator/` directory.
3.  Run the Streamlit application using the command:
    ```bash
    streamlit run app.py
    ```
4.  The application will open in your web browser, allowing you to input numbers, select operations, and view results.
